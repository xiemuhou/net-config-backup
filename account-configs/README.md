# 备份只读账号配置模板

这些模板用于创建网络设备配置备份专用账号。建议账号只用于自动备份，不要和管理员账号混用。

默认账号示例：

```text
用户名: backup
密码: 请替换为强密码
```

使用前请根据设备型号、系统版本和现有 AAA/TACACS/RADIUS 策略确认命令是否适用。

## 文件说明

- `huawei-vrp.txt`: 华为 VRP 设备本地只读账号示例
- `cisco-ios.txt`: Cisco IOS/IOS-XE 本地只读账号示例
- `fortigate.txt`: FortiGate 本地只读管理员账号示例

## 权限建议

- 账号只允许 SSH 登录。
- 禁用 Telnet。
- 密码使用强密码，并定期轮换。
- 如果设备支持源地址限制，只允许备份服务器 IP 登录。
- 备份账号不要授予配置修改权限。
- 建议先在一台测试设备验证备份脚本能读取完整配置，再批量下发。
